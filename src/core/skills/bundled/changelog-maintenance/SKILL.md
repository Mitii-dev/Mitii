# Changelog Maintenance

1. Locate the canonical changelog.
2. Detect its format.
3. Resolve the correct tag or commit range.
4. Aggregate changes deterministically.
5. Group user-facing changes.
6. Exclude internal noise.
7. Preserve historical entries.
8. Apply the smallest patch.
9. Validate Markdown and version ordering.

Support Keep a Changelog, Conventional Changelog, Changesets, Release Please, and custom changelog formats.
